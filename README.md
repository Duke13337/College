# 🏛️ CollegeBook — Агрегатор Бронирования Аудиторий

![Python](https://shields.io)
![FastAPI](https://shields.io)
![Vercel](https://shields.io)

**CollegeBook** — это современная веб-платформа, которая объединяет колледжи в единую экосистему. Студенты могут просматривать свободные кабинеты на интерактивных картах, а преподаватели — бронировать их в один клик.

---

## 🎨 Дизайн и Интерфейс

Сайт выполнен в стиле современных маркетплейсов для максимального удобства.

### 🏠 Главный Хаб (Выбор колледжа)
Здесь пользователь выбирает нужное учебное заведение из списка. Каждая карточка содержит фото корпуса и адрес.

### 📍 Интерактивная карта этажа
Позволяет визуально оценить расположение аудиторий и их текущий статус. Кнопки наложены прямо на чертеж здания.

## 👥 Ролевая модель доступа

Проект реализует жесткое разграничение прав согласно ТЗ:

*   **Студент (Гость):** 🟢 Только просмотр. Не требует авторизации.
*   **Преподаватель:** 🔵 Может бронировать аудитории после входа.
*   **Администратор:** 🔴 Полное управление списком колледжей и пользователями.
*   **Тех. служба:** 🟡 Право закрывать аудитории на ремонт.

## 📦 Инструкция по развертыванию

Если вы хотите запустить проект локально:

1. **Клонируйте проект:**
   ```bash
   git clone https://github.com
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите сервер:**
   ```bash
   python -m uvicorn main:app --reload
   ```

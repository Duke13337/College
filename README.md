# 🏛️ CollegeBook

**CollegeBook** — современная платформа для бронирования аудиторий. Агрегатор, объединяющий несколько колледжей в одну систему с интерактивными картами и разграничением прав доступа.

---

## 🖼️ Галерея проекта

### 🏠 Главная страница
Центральный хаб для выбора учебного заведения. Каждая карточка — это отдельный колледж со своей базой аудиторий.
![Hub](<img width="1905" height="970" alt="image" src="https://github.com/user-attachments/assets/061c4c76-6707-4582-9611-db89837939b8" />
)

### 📍 Карта этажа
Интерактивный план здания. Позволяет выбирать и бронировать аудитории прямо на схеме этажа.
![Map](<img width="1906" height="987" alt="image" src="https://github.com/user-attachments/assets/82c33bee-ea1e-48ee-bd33-2748368020e9" />

---

## 🛠️ Стек технологий
*   **Бэкенд:** FastAPI, SQLAlchemy (SQLite)
*   **Фронтенд:** HTML5, CSS3 (Grid/Flexbox), Jinja2

---

## 👥 Роли пользователей
*   **Студент:** Просмотр списка и карты.
*   **Преподаватель:** Бронирование аудиторий.
*   **Тех. служба:** Управление статусом кабинетов.
*   **Администратор:** Управление колледжами и пользователями.

---

## 🚀 Запуск
1. `pip install -r requirements.txt`
2. `python -m uvicorn main:app --reload`
